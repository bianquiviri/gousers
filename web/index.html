Funciones Principales

Alto Rendimiento

Desarrollado con Go y Gin para una respuesta ultra rápida y eficiente.

ORM Integrado

Persistencia de datos robusta con GORM y migraciones automáticas.

Contenedorizado

Listo para desplegar con Docker y Docker Compose.

Rutas del Proyecto

GET /api/v1/users
Obtiene la lista de todos los usuarios registrados.
POST /api/v1/users
Crea un nuevo usuario en el sistema.
GET /api/v1/users/:id
Muestra los detalles de un usuario específico por ID.
GET /swagger/index.html
Acceso a la documentación interactiva Swagger.
